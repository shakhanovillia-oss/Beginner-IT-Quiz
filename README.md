# Beginner IT Quiz

An interactive Python application designed for absolute beginners to test their general computer knowledge.

## Goal
The main objective of this project is to create a welcoming environment for non-tech users, providing immediate feedback and encouragement based on their performance.

## Features
- **Intuitive Questions:** Global topics that anyone can answer.
- **Dynamic Feedback:** Different congratulatory messages based on the final score.
- **Robust Input:** Handles both lowercase and uppercase answers gracefully.

## How to Play
1. Run the script:
   ```bash
   python quiz.py
